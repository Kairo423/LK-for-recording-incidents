# Incident Management System (IMS)

Внутреннее корпоративное веб-приложение для учета, анализа и управления происшествиями.

## 🎯 Описание

Система предназначена для:
- Учета и управления происшествиями (аварии, несчастные случаи, технические инциденты)
- Аналитики и формирования отчетов
- Управления пользователями и настройками системы

## 🎨 Особенности дизайна

- **Цветовая палитра**: Профессиональная схема с акцентом на красный (#CF1217)
- **Типографика**: Шрифт Inter для чистого и современного вида
- **Компоненты**: Минималистичный дизайн с фокусом на удобство использования

## 🚀 Технологии

- **Frontend**: React 18, TypeScript
- **Стилизация**: Tailwind CSS v4
- **Графики**: Recharts
- **Иконки**: Lucide React
- **Формы**: React Hook Form
- **Анимации**: Motion (Framer Motion)

## 👥 Роли пользователей

### Сотрудник (Employee)
- Просмотр главной страницы
- Просмотр и создание происшествий
- Редактирование профиля

### Руководитель (Manager)
- Все функции сотрудника
- Доступ к разделу "Аналитика"
- Формирование отчетов

### Администратор (Admin)
- Все функции руководителя
- Управление пользователями
- Настройки системы

## 🔐 Демо-аккаунты

Для демонстрации используйте следующие учетные данные (пароль любой):

- **admin@company.com** — Администратор (полный доступ)
- **manager@company.com** — Руководитель (с аналитикой)
- **user@company.com** — Сотрудник (базовый доступ)

## 📱 Основные страницы

### 1. Авторизация
Страница входа с валидацией полей

### 2. Главная (Dashboard)
- Статистические карточки
- Последние происшествия
- Быстрый доступ к основным функциям

### 3. Происшествия
- Список всех происшествий с фильтрами
- Создание/редактирование
- Детальный просмотр

### 4. Аналитика (Manager/Admin)
- Круговые и столбчатые диаграммы
- Динамика по месяцам
- Формирование отчетов

### 5. Администрирование (Admin)
- Управление пользователями
- Настройки системы

### 6. Профиль
- Просмотр и редактирование личных данных

## 💡 Особенности реализации

1. **Роутинг**: Реализован через состояние React (SPA)

4. **Адаптивность**: Полностью адаптивный дизайн
